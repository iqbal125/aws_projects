# AWS IAM Access Analyzer

## Capabilities

AWS Identity and Access Management Access Analyzer provides the following capabilities:

- **External Access Analyzers** - Help identify resources in your organization and accounts that are shared with an external entity.

- **Internal Access Analyzers** - Help identify internal access to selected resources in your organization and accounts.

- **Unused Access Analyzers** - Help identify unused access in your organization and accounts.

- **Policy Validation** - Validates IAM policies against policy grammar and AWS best practices.

- **Custom Policy Checks** - Help validate IAM policies against your specified security standards.

- **Policy Generation** - Generates IAM policies based on access activity in your AWS CloudTrail logs.
